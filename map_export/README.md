# Map Component Files

This folder contains only the essential files needed for the map functionality.

## Files Included:
1. **LeafletMapWithDraw.tsx** - Main map component
2. **supabase_config.ts** - Supabase configuration
3. **MapView.tsx** - Page wrapper (optional)
4. **supabase_setup.sql** - Database schema

## Dependencies:
- leaflet ^1.9.4
- leaflet-draw ^1.0.4
- @supabase/supabase-js ^2.39.0
- @mui/material ^5.15.0

See MAP_INTEGRATION_GUIDE.md for complete setup instructions.
